# Changelog

## [0.4.0] - 2025-04-14

### Added
- Plugin Marketplace distribution — install via `/plugin marketplace add`
- AI-powered image generation for ad creatives (via Pollinations API)
- Guided onboarding with `/setup` command and demo mode
- Campaign templates: Product Launch, Seasonal Sale, Brand Awareness, Content Marketing

## [0.3.0] - 2025-04-13

### Added
- Auto-open HTML preview in browser after content creation
- Feedback loop in `/create-content` — iterate on designs before shipping
- `/cold-outreach` command for cold DM/email generation

### Removed
- Legacy Next.js web app (TheMarketer is now CLI-only)

## [0.2.0] - 2025-04-12

### Added
- Agent loop architecture: Morning Brief, performance analysis, campaign arcs
- Full marketing agency platform: multi-brand support, creative studio, outreach, reports
- Curated influencer database with 40+ real influencers across 12 industries
- 12 industry-specific ad templates based on top-performing real ads
- Adaptive feedback loop and opinionated autopilot mode

## [0.1.0] - 2025-04-11

### Added
- Initial release
- Full marketing platform with 8 feature modules
- Landing page and documentation
